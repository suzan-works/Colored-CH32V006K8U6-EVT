# カラーな評価ボード CH32V006K8U6 Evaluation Board with Color Silkscreen PCB

## 概要
CH32V006K8U6（RISC‑V 32 bit MCU）を搭載した、カラーシルク基板の評価ボードです。WCH-LinkE直結でデバッグや書き込みが可能です。

## 搭載ハード
| 項目 | 内容 |
| --- | --- |
| MCU | **CH32V006K8U6** (QFN32) |
| 電源切替 | スライドSW：5V/3.3V |
| リセット | タクトSW |
| 拡張 | Groveコネクタ (I2C) |
